## Gen Z Slang Translator

An app that translates basic English into Gen Z slang.
Built as a fun project to explore modern internet language.

### Features

Translates plain English phrases into Gen Z slang

Clean, easy-to-use interface

Instant results

Lightweight and fast

### Tech Stack

HTML

CSS

JavaScript


### Usage

Enter a sentence in basic English.

Click the translate button.

Get the Gen Z slang version instantly.

### Purpose

This project was created for pure fun. It is not intended to be a perfect linguistic translator, but rather take on how Gen Z slang is commonly used online.

### Future Improvements

Expand the slang dictionary

Add reverse translation (Gen Z → English)

Improve sentence-level context handling

Mobile UI enhancements
